# 化学公式计算器
一款基于HTML5实现的化学公式计算工具，支持化学方程式配平、分子量计算、摩尔质量计算等功能，提供元素周期表查询、化学反应计算等功能，适配PC与移动端设备。
![](http://debuff.org/img/i/2026/02/13/vz29g0.png)


## 核心功能
1. **化学方程式配平**：自动配平化学方程式，支持复杂反应（60个预设方程式）
2. **摩尔质量计算**：计算化合物的摩尔质量，包含科学介绍（20个预设分子）
3. **元素周期表查询**：查看元素信息（原子量、电子排布等），包含所有元素（118个）
4. **化学反应计算**：计算反应物和生成物的质量关系（20个预设反应）

## 技术栈
- 前端：HTML5、JavaScript、CSS3
- 公式解析：math.min.js（轻量级数学计算库）
- 布局：原生CSS3弹性布局+定位，无第三方UI框架

## 目录结构
```
├── css/
│   └── style.css    # 样式文件，包含所有布局与样式定义
├── js/
│   └── chemistry.js # 核心逻辑文件，化学计算、交互逻辑
├── lib/
│   └── math.min.js  # 数学计算依赖库
└── index.php        # 入口文件，页面结构与DOM定义
```

## 使用方法
1. 克隆项目到本地，确保所有文件目录结构保持不变；
2. 将项目部署到PHP环境或直接通过浏览器打开index.php文件；
3. 展开左侧「化学计算」抽屉，选择计算类型；
4. 输入化学方程式或化合物，点击计算按钮；
5. 查看右侧卡片获取详细计算结果和说明。

## 支持的计算类型
### 化学方程式配平
- 支持酸碱反应、氧化还原反应、沉淀反应等
- 示例：`H2 + O2 -> H2O`、`Fe + CuSO4 -> FeSO4 + Cu`

### 分子量计算
- 支持常见化合物和离子
- 示例：`H2O`、`NaCl`、`C6H12O6`

### 摩尔质量计算
- 基于分子量计算摩尔质量
- 示例：`H2O` 的摩尔质量为 18.015 g/mol

### 元素周期表查询
- 提供常见元素信息
- 包括原子量、电子排布、常见化合价等

## 交互说明
1. 化学计算：左侧抽屉可通过「展开/收起」按钮控制显示/隐藏；
2. 计算结果：选择计算类型后自动显示结果；
3. 说明卡片：右侧卡片可通过关闭按钮隐藏，选择计算类型自动更新说明内容；
4. 设备适配：移动端自适应布局，操作体验优化。

## 注意事项
1. 化学方程式配平功能基于原子守恒原理；
2. 分子量计算使用标准原子量数据；
3. 确保lib目录下的math.min.js文件存在且路径正确；
4. 移动端建议横屏使用，获得更好的展示与操作体验。
